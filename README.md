# Eye Tracking Analysis for Dyslexia Detection

## Описание проекта
Проект использует данные айтрекинга (движений глаз) для автоматического выявления дислексии у детей. Система анализирует паттерны чтения и строит прогностические модели для помощи в ранней диагностике.

## Цели
- **Классификация:** Различить детей с дислексией и нормально читающих сверстников
- **Анализ:** Выявить ключевые различия в паттернах чтения
- **Визуализация:** Наглядно представить стратегии чтения

## Данные
### Источники:
1. **Фиксации глаз** (`fixations.csv`):
   - Координаты взгляда (X, Y)
   - Длительность фиксаций
   - Номер слова в предложении
   - Идентификаторы участников и предложений
   - Предложения

3. **Социодемографические данные** (`socdem.csv`):
   - Возраст, пол, класс
   - Уровень IQ
   - Диагноз (метка)

## Методы анализа

### 1. **Статистический анализ**
- Распределение длительности фиксаций
- Количество фиксаций на предложение
- Анализ регрессивных движений

### 2. **Визуализация паттернов чтения**
- Облака фиксаций для групп детей
- Индивидуальные графы движений глаз
- Сравнительные визуализации

### 3. **Машинное обучение**
- **Последовательные модели:** LSTM/Transformer для временных рядов фиксаций
- **Графовые модели:** GNN для представления структуры чтения
  
## Ключевые признаки

### Временные:
- Средняя длительность фиксаций
- Общее время чтения предложения
- Время первого прохода по словам

### Пространственные:
- Координаты фиксаций
- Амплитуда саккад (движений глаз)
- Пространственный разброс взгляда

### Поведенческие:
- Количество регрессий (возвратов назад)
- Доля рефиксаций (повторных просмотров)
- Пропуски слов

##  Результаты
- AUC: 0.95
- Accuracy: 0.88
- Recall: 0.88
